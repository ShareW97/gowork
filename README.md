# 弈棋无限 · 对弈棋局分析评价

面向围棋教师的本地网页工作台，用于导入 SGF 主分支、逐手复盘、截取关键局面、
提交文字点评与变化图，并生成 PDF 教学报告。

## 一键启动

项目无第三方运行依赖，只需要电脑已安装 Node.js。

- macOS：双击 `start-mac.command`
- Windows：双击 `start-windows.bat`
- Linux / 通用终端：运行 `./start-linux.sh`

启动窗口会显示两个访问地址：

- `http://127.0.0.1:4173`：当前电脑自己访问
- `http://局域网IP:4173`：同一 Wi-Fi / 局域网内的其他电脑、平板访问

请保持启动窗口打开；关闭窗口后，网页会显示断线提示，截图文件也无法继续保存到本地文件夹。

如果喜欢用命令行，也可以直接启动：

```bash
node server.mjs
```

如需指定端口：

```bash
PORT=4174 node server.mjs
```

如只允许本机访问：

```bash
HOST=127.0.0.1 node server.mjs
```

## 局域网访问说明

其他电脑打开启动窗口里显示的 `http://局域网IP:4173` 即可使用。若打不开，请检查：

- 两台设备是否在同一 Wi-Fi / 局域网
- 启动服务的电脑是否允许防火墙放行 Node.js
- 启动窗口是否仍保持打开
- 端口 `4173` 是否被其他程序占用

浏览器页面顶部会显示服务运行状态。连接断开时会出现“重试连接”按钮。

注意：账号与归档数据会统一保存在启动服务那台电脑的 `data/app-data.json` 中，其他电脑访问同一局域网地址会读取同一份数据；浏览器 `localStorage` 只作为本机缓存。点评过程中产生的图片文件会保存到启动服务那台电脑的 `review-images/` 文件夹。

## 功能

- 初始界面提供“棋局分析”和“自由分析”两个入口
- 点击上传或拖拽导入 `.sgf` 棋谱，也可载入内置示例
- 解析并重建 SGF 主分支，支持提子、停一手和初始摆子
- 填写对弈时间、平台与结果
- 自由分析支持 9 路、13 路、19 路棋盘切换
- 自由分析支持计算解析、定式变化、全局分析三种报告类型，并在 PDF 首页展示不同信息
- 自由分析支持交替落子、添加黑子、添加白子三种方式；添加棋子不计入手数，也不显示最后一手标记
- 自由分析清空棋盘时，可选择是否同步清空右侧点评归档
- 归档内容页按棋局分析、计算分析、定式变化、全局分析四类整理并按时间排序
- 使用前进、后退和进度条精准定位每一步
- 截取当前局面，提交文字点评与多张带编号的教师变化图
- 主棋盘与变化图均支持数字、字母、三角和圆圈标记
- 每张变化图可单独填写文字点评
- 变化图可通过“倒退一手”调整到主分支前一手作为推演起点
- 点评过程中产生的局面截图与变化图会保存到本地 `review-images/` 文件夹
- 从点评归档点击“分析”，继续修改文字及增删变化图
- 生成高清、紧凑排版的图文 PDF 教学报告
- 在浏览器本地保存棋局点评并继续编辑
- 将对局信息、全部点评、截图和变化图整理为 PDF 报告

## 测试

```bash
node --test
```

共享账号与归档数据保存在启动服务电脑的 `data/app-data.json` 中；浏览器本地 `localStorage` 仅用于缓存与会话登录状态。
